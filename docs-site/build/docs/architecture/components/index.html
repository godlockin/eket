组件概览

EKET 核心组件介绍。

核心组件​

1. Connection Manager​

四级降级连接管理：

Remote Redis → Local Redis → SQLite → File

功能:

  • 自动检测可用连接
  • 优雅降级
  • 连接池管理

2. Message Queue​

消息队列（三级降级）：

  • Level 3: Redis Pub/Sub
  • Level 2: 优化文件队列
  • Level 1: 基础文件队列

特性:

  • ✅ 原子写入
  • ✅ 校验和验证
  • ✅ 自动去重
  • ✅ 重试机制

3. Circuit Breaker​

断路器模式：

const circuit = new CircuitBreaker({
  maxFailures: 5,
  resetTimeout: 30000
});

状态:

  • closed - 正常
  • open - 熔断
  • half_open - 半开（测试恢复）

4. Cache Layer​

LRU 缓存 + Redis 二级缓存：

const cache = new CacheLayer({
  useRedis: true,
  maxMemorySize: 100 * 1024 * 1024
});

特性:

  • 内存缓存 (Level 2)
  • Redis 缓存 (Level 3)
  • 缓存穿透保护

5. Master Election​

三级 Master 选举：

  • Redis SETNX (Level 3)
  • SQLite 行锁 (Level 2)
  • 文件 mkdir (Level 1)

6. Knowledge Base​

知识库系统（Level 3）：

  • Retrospective - 回顾
  • Pattern - 模式
  • Decision - 决策
  • Lesson - 经验

7. Event Bus​

事件总线：

  • DomainEvent - 领域事件
  • 死信队列 - 失败事件处理

8. Alerting​

四级告警：

  • info - 信息
  • warning - 警告
  • error - 错误
  • critical - 严重

辅助组件​

9. Optimized File Queue​

优化文件队列：

  • 临时文件 + rename
  • 校验和验证
  • 自动归档

10. Instance Registry​

实例注册与心跳管理

11. Agent Pool​

Agent Pool 管理：

  • 负载均衡
  • 角色选择

12. Workflow Engine​

工作流引擎：

  • 预定义协作流程
  • 状态机管理
