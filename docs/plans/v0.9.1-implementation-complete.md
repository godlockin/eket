### 2. Master 选举竞态条件

## 实现方案

### 1. 四级降级策略 ✅

**文件**: `node/src/core/connection-manager.ts` (320 行)

**实现功能**:
- 四级连接降级：`remote_redis` → `local_redis` → `sqlite` → `file`
- 自动初始化：选择最高可用级别
- 统计信息追踪
- 支持升级检测

**核心代码**:
```typescript
export class ConnectionManager {
  async initialize(): Promise<Result<ConnectionLevel>>;
  getCurrentLevel(): ConnectionLevel;
  getStats(): ConnectionStats;
  tryUpgrade(): Promise<Result<ConnectionLevel>>;
}
```

**配置支持**:
```typescript
{
  remoteRedis?: { host, port, password, db };
  localRedis?: { host, port };
  sqlitePath?: string;
  fileQueueDir?: string;
  driverMode: 'js' | 'shell';
}
```

---

### 2. Master 选举机制 ✅

**文件**: `node/src/core/master-election.ts` (650 行)

**实现功能**:
- 三级选举：Redis SETNX → SQLite 事务 → File mkdir（原子操作）
- 分布式锁：防止竞态条件
- 声明等待期（2 秒）：检测冲突
- 租约机制（30 秒）：自动续期（每 15 秒）
- 降级逻辑：高级别不可用时自动降级

**核心代码**:
```typescript
export class MasterElection {
  async elect(): Promise<Result<MasterElectionResult>>;
  isMasterNode(): boolean;
  getInstanceId(): string;
  async relinquish(): Promise<void>;
  async close(): Promise<void>;
}
```

**选举流程**:
```
1. 尝试获取分布式锁 (SETNX/INSERT/mkdir)
2. 获取成功后进入声明等待期 (2 秒)
3. 等待期内检测是否有其他 Master 声明
4. 无冲突则正式成为 Master
5. 创建 Master marker 文件
6. 启动租约续期
```

## 技术亮点

### 1. 分布式锁实现
- **Redis**: `SET key value NX PX timeout`
- **SQLite**: `INSERT OR IGNORE` + 事务
- **File**: `fs.mkdir(path, { recursive: false })` 原子操作

### 2. 防竞态设计
```
Instance A: 获取锁 → 声明等待期 → 检测冲突 → 成为 Master
Instance B: 获取锁失败 → 检测锁 → 返回 Slaver
```

### 3. 租约续期机制
```typescript
// 每 15 秒自动续期，防止 Master 意外宕机导致系统停滞
private startLeaseRenewal(level: ElectionLevel, lockFile?: string) {
  this.leaseTimer = setInterval(async () => {
    await this.renewLease(level, lockFile);
  }, this.config.leaseTime / 2);
}
```

### 4. 四级降级自动检测
```typescript
async initialize(): Promise<Result<ConnectionLevel>> {
  // 1. 尝试远程 Redis
  if (config.remoteRedis) {
    const result = await tryRemoteRedis();
    if (result.success) return 'remote_redis';
  }
  // 2. 尝试本地 Redis
  if (config.localRedis) {
    const result = await tryLocalRedis();
    if (result.success) return 'local_redis';
  }
  // 3. 尝试 SQLite
  if (config.sqlitePath) {
    const result = await trySQLite();
    if (result.success) return 'sqlite';
  }
  // 4. 降级到文件
  return 'file';
}
```

## 使用示例

### ConnectionManager
```typescript
import { createConnectionManager } from './core/connection-manager.js';

const manager = createConnectionManager({
  remoteRedis: { host: 'redis-cluster', port: 6379 },
  localRedis: { host: 'localhost', port: 6379 },
  sqlitePath: '~/.eket/eket.db',
  fileQueueDir: './.eket/queue',
});

const result = await manager.initialize();
console.log(`当前级别：${manager.getCurrentLevel()}`);
console.log(`统计：${JSON.stringify(manager.getStats())}`);
```

### MasterElection
```typescript
import { createMasterElection } from './core/master-election.js';

const election = createMasterElection({
  redis: { host: 'localhost', port: 6379 },
  projectRoot: '/path/to/project',
  electionTimeout: 5000,
  declarationPeriod: 2000,
  leaseTime: 30000,
});

const result = await election.elect();
if (result.success && result.data.isMaster) {
  console.log('成为 Master！');
} else {
  console.log('成为 Slaver');
}
```
