## 🔄 审查周期

### 建议的审查频率

| 审查类型 | 频率 | 说明 |
|---------|------|------|
| **全面审查** | 每季度一次 | 全面检查所有文档 |
| **快速审查** | 每月一次 | 检查核心文档和新文档 |
| **持续监控** | 每周一次 | 检查新创建/修改的文档 |
| **版本审查** | 每次发版 | 确保版本号一致性 |
