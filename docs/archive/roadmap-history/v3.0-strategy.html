🔄 Round 13b 规划详情

GitHub Actions CI

name: CI
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: cd node && npm ci
      - run: cd node && npm run build
      - run: cd node && npm test

健康检查端点

  • GET /health - 实例健康状态
  • GET /ready - 是否准备好接收请求
  • GET /live - 存活探针

分支保护规则

  • main: 保护，需要 PR + 审核 + CI 通过
  • testing: 保护，需要 CI 通过
  • feature/*: 开放，Slaver 开发使用
