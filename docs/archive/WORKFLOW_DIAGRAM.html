通信协议图

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              三种通信方式                                        │
└─────────────────────────────────────────────────────────────────────────────────┘

  1. 消息队列（实时通信）
  ┌─────────────────────────────────────────────────────┐
  │  shared/message_queue/                              │
  │  ├── inbox/          # 入队消息                      │
  │  │   ├── msg_001.json                               │
  │  │   └── msg_002.json                               │
  │  ├── outbox/         # 出队消息                      │
  │  │   ├── msg_003.json                               │
  │  │   └── msg_004.json                               │
  │  └── broadcast/      # 广播消息                      │
  │      └── announcement_001.json                      │
  └─────────────────────────────────────────────────────┘

  2. Git Commit（状态持久化）
  ┌─────────────────────────────────────────────────────┐
  │  Git 仓库提交历史                                    │
  │  ├── feat(auth): add login form     [task-001]      │
  │  ├── test(auth): add login tests    [task-001]      │
  │  └── chore: update task status      [task-001]      │
  └─────────────────────────────────────────────────────┘

  3. 共享状态文件（跨实例同步）
  ┌─────────────────────────────────────────────────────┐
  │  shared/.state/                                     │
  │  ├── agent_states.json     # 智能体状态              │
  │  ├── task_assignments.json # 任务分配表              │
  │  └── locks/              # 文件锁                    │
  │      └── config.yml.lock                            │
  └─────────────────────────────────────────────────────┘

│                              人类反馈循环                                        │

    ┌──────────────┐

    │  一轮工作     │

           │
           ▼
    ┌──────────────────────────────────┐
    │ 创建状态报告                       │
    │ inbox/human_feedback/            │
    │ status-<task-id>-<timestamp>.md  │
    └──────┬───────────────────────────┘
           │
           │ 状态：pending_confirmation
           ▼
    ┌──────────────┐
    │  等待人类确认  │
    │  (智能体暂停)  │
    └──────┬───────┘
           │
           │ 人类回复
           ▼
    ┌──────────────────────────────────┐
    │ 人类在同一个文件中回复             │
    │ - 确认成果                        │
    │ - 选择推荐答案                    │
    │ - 提供额外指示                    │
    └──────┬───────────────────────────┘
           │
           ▼
    ┌──────────────┐
    │  智能体读取   │
    │  人类反馈     │
    └──────┬───────┘
           │
           │ 根据反馈

    ┌──────────────────────────────────┐
    │ 继续执行或修改                    │
    │ - 进入下一阶段                    │
    │ - 调整方案重新执行                │
    └──────────────────────────────────┘
