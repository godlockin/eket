后续维护建议

  1. 定期清理测试结果: tests/results/ 每次测试后生成
  2. 保持文档同步: 更新 docs/ 时删除旧版本
  3. 模板保持精简: template/ 仅包含必需文件
  4. 状态文件清理: .eket/state/ 定期清理
