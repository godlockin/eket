# 归档文档

**最后更新**: 2026-04-15

本目录存放 EKET 框架的历史文档，仅供参考。**新用户请直接阅读 [docs/](../README.md)。**

---

## 目录结构

```
archive/
├── v0.2/           # v0.2 版本文档
├── v0.5/           # v0.5 版本文档
├── v0.6/           # v0.6 版本文档
├── audit-history/  # 历史审计报告
├── plans/          # 历史设计方案
├── status-history/ # 历史状态快照
└── *.md            # 跨版本历史文档
```

## 归档标准

以下情况的文档归入此目录：

1. **版本过时** — 描述已弃用版本的功能
2. **已被取代** — 有更新版本的同类文档
3. **计划已执行** — 原始设计方案，功能已实现
4. **历史快照** — 某时间点的状态记录，无持续指导意义
