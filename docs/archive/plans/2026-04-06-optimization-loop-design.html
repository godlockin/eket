循环架构

优化循环控制器（用户确认每层进入下一层）
         │
    ┌────▼────┐     ┌────▼────┐     ┌────▼────┐
    │ Layer 1 │ --> │ Layer 2 │ --> │ Layer 3 │
    │ Bug修复 │     │架构整理 │     │能力提升 │
    └─────────┘     └─────────┘     └─────────┘

每层内部：
  域A升级Agent ──┐
                 ├──► 评判Agent ──► 通过? ──► 用户确认进下层
  域B升级Agent ──┤                    │
  域C升级Agent ──┘          不通过 ──► 定向修正轮
