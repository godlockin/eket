# EPIC-015: 内容生产流水线

## 概述

将 14 篇 EKET 技术文章转化为多平台完整素材：
- 小红书图文/视频
- 微信公众号图文
- (后续) 抖音/TikTok/YouTube 视频

## 目标

1. 搭建半自动内容生产流水线
2. 统一视觉风格（封面模板+火柴人演绎）
3. 产出 14 套可发布素材

## 技术栈

- TTS: Edge TTS (免费)
- 封面: Canva 模板
- 火柴人: OpenPeeps/手绘 SVG
- 视频: FFmpeg + Python
- 字幕: pysrt

## 视觉风格

- 封面：深色渐变 + 白色标题 + L1-L4 徽章
- 视频：PPT 解说 + 火柴人演绎
- 统一但每篇微调
