# EKET 反模式库

## 1. 隐藏假设 (Hidden Assumptions)

**场景**: "添加用户数据导出功能"

❌ **反模式**:
直接实现 CSV 导出，未确认格式、字段、权限。

✅ **正确做法**:
先列出假设并确认：
- 导出格式？(CSV/JSON/Excel)
- 包含哪些字段？敏感字段如何处理？
- 谁有权限导出？需要审计日志吗？

---

## 2. 过度抽象 (Over-Abstraction)

**场景**: "写一个计算折扣的函数"

❌ **反模式**:
```python
class DiscountStrategyFactory:
    def create_strategy(self, type: str) -> DiscountStrategy:
        ...
```

✅ **正确做法**:
```python
def calculate_discount(price: float, percent: float) -> float:
    return price * (1 - percent / 100)
```
