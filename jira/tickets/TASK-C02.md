# TASK-C02: 封面模板设计

- **EPIC**: EPIC-015
- **优先级**: P0

## 描述

设计统一的封面模板，支持 14 篇文章微调复用。

## 设计规范

### 尺寸
- 小红书: 1080x1440 (3:4) 或 1080x1080 (1:1)
- 公众号: 900x383 (推送封面)
- 视频封面: 1080x1920 (9:16)

### 统一元素
- **背景**: 深色渐变 `#1a1a2e → #16213e`
- **标题**: 白色粗体，居中
- **副标题**: 浅蓝色 `#4fc3f7`，小号
- **Logo**: 右下角 EKET 水印
- **系列徽章**: 左上角

### 徽章颜色
| 系列 | 颜色 | Hex |
|------|------|-----|
| L1 入门 | 绿色 | #4caf50 |
| L2 模块 | 蓝色 | #2196f3 |
| L3 决策 | 紫色 | #9c27b0 |
| L4 教训 | 橙色 | #ff9800 |

### 微调变量
- 徽章颜色 (按系列)
- 装饰图标 (按主题)
- 火柴人姿态 (按内容)
