# EKET 封面模板规范

> 统一视觉风格，14 篇文章复用

## 设计规范

### 尺寸标准

| 平台 | 尺寸 | 比例 | 用途 |
|------|------|------|------|
| 小红书 | 1080x1440 | 3:4 | 笔记封面 |
| 公众号 | 900x383 | 2.35:1 | 推送封面 |
| 视频 | 1080x1920 | 9:16 | 视频号/抖音 |

### 色彩系统

**背景渐变**
```
起点: #1a1a2e (深紫蓝)
终点: #16213e (深蓝)
方向: 垂直 0° → 100%
```

**文字颜色**
- 标题: `#FFFFFF` (白色粗体)
- 副标题: `#4fc3f7` (浅蓝)
- 水印: `#666666` (灰色)

**系列徽章**
| 系列 | 颜色 | Hex | 适用文章 |
|------|------|-----|----------|
| L1 入门 | 绿色 | #4caf50 | 01-03 |
| L2 模块 | 蓝色 | #2196f3 | 04-07 |
| L3 决策 | 紫色 | #9c27b0 | 08-11 |
| L4 教训 | 橙色 | #ff9800 | 12-14 |

### 排版规范

**小红书 1080x1440**
- 徽章: 左上角 (40, 40)，80x40px，圆角 8px
- 标题: 居中 (540, 700)，56px 粗体
- 副标题: 居中 (540, 780)，32px
- Logo: 右下角 (1000, 1400)，24px

**公众号 900x383**
- 徽章: 左上角 (30, 30)，70x35px，圆角 6px
- 标题: 居中 (450, 180)，40px 粗体
- 副标题: 居中 (450, 230)，24px
- Logo: 右下角 (850, 360)，18px

## 模板文件

| 文件 | 说明 |
|------|------|
| `template-xiaohongshu.svg` | 小红书封面模板 |
| `template-wechat.svg` | 公众号封面模板 |

## 使用方法

1. 复制对应 SVG 模板
2. 替换占位符:
   - `{{TITLE}}` → 文章标题
   - `{{SUBTITLE}}` → 副标题
   - `L1` / 徽章颜色 → 按系列调整
3. 导出 PNG (2x 清晰度)

## 示例

```
L1 入门: 徽章 #4caf50，标题"什么是 EKET？"
L2 模块: 徽章 #2196f3，标题"核心模块详解"
L3 决策: 徽章 #9c27b0，标题"架构决策分析"
L4 教训: 徽章 #ff9800，标题"踩坑经验总结"
```
