## 输入规范说明

每条需求遵循以下结构：

```
[优先级标注] 标题
时间：YYYY-MM-DD HH:MM

### 背景
当前状态是什么？为什么要做这件事？

### 目标
做完之后，什么改变了？

### 约束条件（可选）
技术栈、时间限制、不能动的模块……

### 期望产出（可选）
具体的文件、功能、指标……

### 参考资料（可选）
相关链接、文档路径……
```
